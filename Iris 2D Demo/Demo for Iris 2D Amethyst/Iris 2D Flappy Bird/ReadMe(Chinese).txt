Iris 2D Amethyst Flappy Bird Demo
版本号：1.0.0.0
发布时间：2014.11.9

介绍：使用Iris 2D开发的一个简单功能的Flappy Bird Demo，仅作Iris 2D演示用，游戏素材搜集自网络。
文档说明：

Game 文件夹下为游戏本体，点开 Iris 2D Flappy Bird.exe 即可进入游戏。若无法正常启动游戏，请检查您的环境是否安装了 Microsoft Visual C++ 2010/2012/2013 运行库。
Source Code 文件夹下为游戏的所有源码以及所需要的素材，您可以自行编译源码运行。
本源码附属于 Iris 2D Amethyst ，遵守 Iris 2D Project 所基于的一切开源协议以及于Iris 2D中附加规定的内容。
详情请参考 Iris 2D Project。
